    How do I use this map?

    - Select layers with icon in upper right corner
    - Click on items to get info including location, time and event number
    - Subset times using the calendar fields at the top of the page. Note, this will reset the map, so select time before selecting layers
    - If you don't see any background imagery, visit https://dasng.nbp.usap.gov click "advanced" and "proceed to dasng.nbp.usap.gov (unsafe)" or similar, then come back here and refresh the page. This will tell your browser to trust data coming from NBP's servers
    - You can use the ruler on the left to get a distance between points. Do the same with https://maps.nbp.usap.gov to get map layers
    - Points of interest can be added to the map by editing a file on the science drive science/NBP2202/data-to-map/points-for-map.txt. You can right-click on the map and select "show coordinates" where you wish to place a pin. NB These points can take up to a minute appear on the map & will require a refresh

    Where are the data coming from?

    - Ships's track and several background layers like place names and GEBCO bathymetry are pulled from the ship's systems in the same manner as the NBP map
    - AMSR ice extent and MODIS imagery are created from the AMSR and MODIS geotiffs using gdal2tiles (thanks Ashley!) and gdalwarp (thanks Li!). This is updated daily
    - Glider data are pulled from a database that is updated a few times a day
    - Other datasets are automagically parsed from the event log. You can see the scripts used to perform this on the science drive in "data-to-map"
    - Want to see your dataset on here? Talk to Callum :)

    What else is here?

    - This website is based on the ueaglider website with modifications to pull data from NBP rather than the internet
    - All code is available in a GitHub repo or from Callum
    - This is a work in progress! Any feedback is much appreciated
